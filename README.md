# [signature.html](https://dudushy.github.io/signature.html/)
Trying to create a signature pad using HTML Canvas.

---
## Prerequisites:
- [NodeJS](https://nodejs.org/)

## How to install:
1. Install packages
    ```bash
    npm i
    ```
2. Run project
    ```bash
    npm start
    ```
3. Build project
    ```bash
    npm run build
    ```
